Bonjour, ceci est le premier fichier crée dans le cadre de l'activité de la 2ème partie du cours sur Git et GitHub.

Ce projet a pour but de mettre en pratique l'ensemble des actions possibles avec Git et GitHub. 
Les consignes sont de créer un repository Git sur notre machine locale. Ensuite de créer un fichier texte README.txt (ce que vous êtes en train de lire). 
Par la suite, insérer 2 autres fichiers de notre choix et créer au moins 4 commits différents afin de voir le cheminement.

Voici ma marche à suivre :
	- J'ai créé un repository "Activité_Partie_2".
	- J'ai ensuite créé un fichier texte "README.txt".
	- J'ai réalisé un premier commit nommé "Création du fichier README.txt : explication du projet".
	- J'ai crée un deuxième fichier "cours_html".
	- J'ai réalisé un deuxième commit nommé "Ecriture du cours sur HTML"
	- J'ai créé un troisième fichier nommé "cours_php"
	- J'ai réalisé un troisième commit nommé "Ecriture du cours sur PHP"
	- J'ai modifié le fichier "cours_html".
	- J'ai réalisé un quatrième commit nommé "Correction orthographique du cours sur HTML"
	
Ensuite, j'ai envoyé le ropository "Activité_Partie_2" sur GitHub.
